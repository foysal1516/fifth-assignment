What is the Document Object Model?(DOM)
The Document Object Model (DOM) is a programming interface that represents an HTML or XML document as a structured tree of objects. It allows scripts (like JavaScript) to dynamically manipulate the content, structure, and styles of a webpage. Each element, attribute, and text in the document is a node in the DOM tree, which can be accessed and modified using methods like getElementById, innerHTML, and appendChild.

What is event delegation in the context of DOM?and why is it useful?
Event delegation is a technique in the DOM where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, you attach a single listener to their common ancestor.
Why is it useful?
Improves Performance – Reduces memory usage by adding fewer event listeners. Handles Dynamically Added Elements – Works even if new child elements are added later. Simplifies Code – Avoids adding and removing multiple listeners manually.
